# Design Concept

## Theme
- **Vibe**: "AI", "Futuristic", "Generative".
- **Visuals**:
    - Dark mode default.
    - Particle systems or neural network visualizations on canvas.
    - Smooth scrolling with parallax effects.
    - Glassmorphism for cards and overlays.

## Key Elements
1.  **Hero Section**:
    - Full-screen canvas background (e.g., flowing particles, connecting nodes).
    - Large, bold typography.
    - "Scroll to explore" indicator.
2.  **Projects**:
    - Interactive cards.
    - Hover effects revealing details.
3.  **Admin**:
    - Minimalist interface.
    - Focus on functionality.
