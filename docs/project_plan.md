# Project Implementation Plan

## Overview
A personal portfolio website built with Next.js, featuring a modern, AI-inspired aesthetic and an admin panel for content management.

## Tech Stack
- **Framework**: Next.js (App Router)
- **Language**: TypeScript
- **Styling**: Vanilla CSS (CSS Modules)
- **Animation**: Framer Motion (planned)
- **Canvas**: HTML5 Canvas API for "AI vibe" background
- **Data Storage**: JSON files (initially) for easy portability and editing via Admin.

## Phases
1.  **Setup**: Initialize Next.js project.
2.  **Core UI**: Implement the main layout and "AI vibe" canvas background.
3.  **Components**:
    - Hero Section (Short & Punchy intro)
    - Project Showcase (Grid/List of programs)
    - Admin Login
    - Admin Dashboard (Edit content)
4.  **Admin Feature**:
    - Secure route (`/admin`)
    - Read/Write to content files.
5.  **Polish**: Animations, SEO, Performance.
